
# Student Result Management System

A beginner-friendly Python project that calculates a student's result based on marks out of 300.

## Features

- Calculate percentage
- Assign grade automatically
- Generate remarks
- Determine PASS/FAIL status
- Save results to a text file
- Display all saved results

## Technologies Used

- Python 3

## How to Run

1. Clone the repository
2. Open a terminal
3. Run:

```bash
python result.py
```

## Example Output

```
Enter your name: Sagar
Enter total marks (out of 300): 270

Percentage: 90.00%
Grade: A+
Remark: Outstanding

Result saved successfully!
```
